p.md2Archbee

Dieses Tool optimiert aus Paligo exportierte Markdown-Dateien für den Import in Archbee.

So geht's

  1. Klicken Sie auf Datei auswählen.
  2. Wählen Sie eine zu optimierende Markdown-Datei aus.
  3. Klicken Sie auf Markdown-Datei anpassen.

Die Datei wird optimiert und anschließend automatisch heruntergeladen.

Die Optimierung läuft direkt im Browser – es werden keine Daten gespeichert.
